# jang1563.github.io

Source for the static portfolio site at **https://jang1563.github.io**.

Single-file `index.html` — vanilla HTML + CSS, no build step, no JS dependencies. Dark/light themes follow `prefers-color-scheme`. Hosted on GitHub Pages from the `main` branch.

## Sections

1. Reviewer route in 15 minutes
2. Empirical calibration gap (April 2026)
3. Anchor portfolio (6 repositories with `SAFETY.md` + AI-safety mapping)
4. Evaluation infrastructure
5. Spaceflight biology foundation
6. Active research (May 2026 submissions)
7. Awards & compute grants
8. Service & reviewing

## Updating

Edit `index.html` directly. Changes pushed to `main` deploy in ~30 seconds.
