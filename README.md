# jang1563.github.io

Source for the static portfolio at **https://jang1563.github.io**.

The site presents a capability-first research portfolio across computational biology,
biological AI evaluation, reproducible benchmarks, and spaceflight omics.

## Selected work

1. `grounding-atlas` — content grounding and capability placement
2. `verify-or-trust` — verification allocation for biology agents
3. `GeneLab_benchmark` / SpaceBio-Bench — mission-held-out generalization
4. `agentic-drug-discovery-system` — auditable clinical-trial decision benchmark
5. `LabCraft-Eval` — trajectory-level laboratory agent evaluation

`SpaceOmicsBench` anchors the spaceflight multi-omics research section.

## Implementation

- Single-file `index.html`
- Vanilla HTML and CSS; no build step or JavaScript dependency
- Responsive layout, reduced-motion support, and dark/light themes
- Hosted with GitHub Pages from the `main` branch

## Updating

Edit `index.html` directly. Keep project claims tied to public repository or dataset
artifacts, and preserve explicit scope notes for work that remains partial.
